## 三、文件命名规范

1. **战队Logo**: `team_{序号}_logo.{ext}` 或 `{战队ID}.png`
   - 示例: `team_00_logo.png`, `4457.png`

2. **选手头像**: `player_{序号}_avatar.{ext}` 或 `{选手ID}.png`
   - 示例: `player_00_avatar.png`, `26358.png`

3. **英雄图标**: `hero_{英雄ID}.jpg`
   - 示例: `hero_501.jpg`, `hero_105.jpg`

4. **城市地标**: `{城市拼音}_{地标拼音}.jpg`
   - 示例: `beijing_gugong.jpg`, `chengdu_panda.jpg`

5. **美食图标**: `food_{城市拼音}_{美食拼音}.jpg`
   - 示例: `food_beijing_kaoya.jpg`

6. **UI元素**: `{类型}_{名称}_{状态}.{ext}`
   - 示例: `btn_primary_normal.png`, `panel_main.png`

### 5.1 kpl_image_scraper.py
从KPL官网抓取战队Logo和选手头像

**使用方法**:
```bash
pip install playwright
playwright install chromium
python kpl_image_scraper.py
```

### 5.2 download_heroes.py
从KPL官网CDN下载英雄图标

**使用方法**:
```bash
python download_heroes.py
```

### 5.3 download_images.py

**使用方法**:
