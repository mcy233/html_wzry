## 一、游戏概述

### 1.1 一句话描述

玩家选择一支KPL战队，代表其所在城市，从常规赛一路披荆斩棘，最终征战年度总决赛、夺取冠军奖杯的电竞策略冒险游戏。

### 1.2 核心卖点

| 卖点 | 说明 |
|------|------|
| **未来赛季，你来书写** | 时间线设定在2027年KPL年度总决赛——一个尚未发生的赛季，结局由玩家决定 |
| **18城18队，各具特色** | 真实KPL战队 × 中国城市文化，每个选择都是截然不同的体验 |
| **策略对战 + 剧情探索** | 不是简单的猜拳比赛，而是有Ban/Pick策略、阵容搭配、选手状态管理的深度玩法 |
| **城市文旅融合** | 每座城市都有独特的地标场景、文化彩蛋、美食收集，电竞与文旅深度结合 |

### 1.3 目标用户

- 王者荣耀 / KPL 电竞爱好者
- 对城市文旅感兴趣的年轻用户
- 喜欢策略经营类游戏的玩家

## 二、世界观与故事背景

### 2.1 故事背景

> 2027年，KPL联赛迎来了最令人瞩目的一个赛季。各大战队完成了大规模的阵容重组，新生代选手崭露头角，老将们也燃起了再战巅峰的斗志。从春季赛到夏季赛，再到年度总决赛——这一年的KPL，注定不平凡。

> 而你，将以战队经理的身份，带领一支KPL战队，从城市主场出发，征战全国，向着年度总决赛的冠军奖杯发起冲击。

> 你的选择，将决定这个赛季的最终结局。

### 3.2 一个典型游戏日的流程

```
【比赛日流程示例：春季赛第一轮 第2场】

1. 晨间训练（小游戏）
   → 点击节奏小游戏，训练选手手速
   → 成功：选手当日操作属性+5%

2. 赛前准备
   → 查看今日对手：重庆狼队
   → 情报显示对方近期偏好"4-1分推"战术
   → 选择应对战术：团战流 / 运营流 / 入侵流

3. 阵容确认
   → 从大名单中选择首发5人
   → 注意选手状态：暖阳（状态良好）、小麦（轻微疲劳）
   → 确认首发阵容

4. 比赛进行（BO3）
   → 第一局：Ban/Pick → 战斗演出 → 胜利！
   → 第二局：Ban/Pick → 战斗演出 → 失败...
   → 第三局：触发关键抉择 → 选择决胜战术 → 胜利！

5. 赛后
   → 获取积分、经验值、金币
   → 选手疲劳度增加
   → 解锁新的城市探索事件
```

## 五、详细游戏机制

### 5.1 属性系统

#### 选手属性（6维）

```
        操作 (Operation)
          ╱╲
         ╱  ╲
意识 ───╱    ╲─── 对线
(Sense)╲    ╱(Laning)
        ╲  ╱
         ╲╱
        配合 (Teamwork)
         │
    抗压 ─┼─ 心态
 (Pressure) (Mental)
```

| 属性 | 说明 | 影响 |
|------|------|------|
| 操作 | 选手的微操水平 | 影响1v1对拼、技能释放精度 |
| 意识 | 对战局的判断力 | 影响地图控制、支援时机 |
| 对线 | 线上对抗能力 | 影响前期发育、压制力 |
| 配合 | 与队友的默契度 | 影响团战协同、连招衔接 |
| 抗压 | 逆风局的发挥 | 影响落后时的能力衰减幅度 |
| 心态 | 心理素质 | 影响关键局、大赛的发挥稳定性 |

#### 选手状态系统

```
状态值（0-100）

100 ██████████ 巅峰状态（全属性+10%）
 80 ████████░░ 良好状态（正常发挥）
 60 ██████░░░░ 一般状态（属性-5%）
 40 ████░░░░░░ 疲劳状态（属性-15%）
 20 ██░░░░░░░░ 低迷状态（属性-25%，需要休息）

影响因素：
  ↑ 提升：休息、城市探索、美食、训练成功
  ↓ 降低：连续比赛、训练过度、比赛失败
```

#### 团队属性

| 属性 | 说明 | 提升方式 |
|------|------|---------|
| 士气值 | 全队信心，影响比赛发挥 | 胜利+10，失败-8，城市活动+5 |
| 默契度 | 阵容配合程度 | 同一阵容连续出场每场+2 |
| 声望值 | 战队在联盟中的威望 | 击败强队+大量，败给弱队-大量 |
| 粉丝数 | 影响主场优势 | 比赛胜利、城市互动提升 |

#### Ban/Pick 系统

这是比赛的核心策略环节，简化但保留策略深度：

```
Ban/Pick流程（简化版）：

第一阶段 - Ban（禁用）
  ├── 我方禁用 2 个英雄
  └── 对方禁用 2 个英雄（AI根据对方战术偏好选择）

第二阶段 - Pick（选择）
  ├── 我方选择 3 个英雄
  └── 对方选择 3 个英雄

第三阶段 - 最终选择
  ├── 我方选择 2 个英雄
  └── 对方选择 2 个英雄

策略要点：
  - 需要了解对方选手的英雄池（情报系统）
  - 针对性Ban掉对方核心选手的擅长英雄
  - 阵容搭配需要兼顾前中后期
```

#### 英雄体系（简化分类）

#### 比赛演出系统（核心交互设计）

> 设计理念：不做3D战斗画面，而是把整场比赛做成一个**「KPL转播台控制面板」**。

> 你看到的是和KPL直播类似的数据面板、地图态势、经济曲线，你做的是战术决策。

##### 势能条系统（贯穿全局的视觉锚点）

```
  - 一个div做底色渐变

  - 完全不需要Canvas，纯CSS即可实现
```

### 5.4 过渡与间章系统

```
  → 可能有选手租借事件（类似真实KPL租借机制）

  → 解锁城市的隐藏区域
  → 深度文化体验事件
  → 团队建设活动

Day 11-14: 夏季赛备战
  → 针对性训练（根据春季赛暴露的问题）
  → 新战术研发
  → 新版本适应训练
```

### 6.1 主线剧情结构

每支战队共享相同的赛季框架，但拥有独特的故事线：

```
序章：「启程」
  ├── 战队选择 → 城市初印象
  ├── 认识队员 → 了解每人的故事
  └── 首场训练赛 → 教程关卡

第一章：「春季赛 · 征程」
  ├── 常规赛三轮循环 → 积分排名
  ├── 城市主场氛围 → 主场优势
  └── 关键剧情：队内矛盾/磨合事件

第二章：「春季季后赛 · 冲冠」
  ├── 淘汰赛高压对决
  ├── 关键剧情：选手伤病/状态波动
  └── 春季赛结局（冠军/其他名次）

间章：「休赛期 · 蛰伏」
  ├── 转会窗口 → 阵容调整
  ├── 城市深度探索 → 文化感悟
  └── 关键剧情：选手个人成长故事

第三章：「夏季赛 · 再战」
  ├── 新赛季开始 → 格局变化
  ├── 关键剧情：老对手的进化
  └── 夏季赛征程

第四章：「夏季季后赛」
  ├── 更高强度的淘汰赛
  └── 积分争夺战 → 年终排名

间章：「年总备战 · 磨砺」
  ├── 回顾全年征程
  ├── 关键剧情：最终的决心
  └── 选手状态巅峰调整

终章：「年度总决赛 · 荣耀」

  ├── 擂台赛阶段
  ├── 巅峰对决 → BO7终极之战
  └── 结局：冠军加冕 / 遗憾落幕 / 明年再来
```
