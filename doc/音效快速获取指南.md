## 三、音效处理规范

### 3.1 格式转换

下载的音效可能需要转换格式，使用 **Audacity**：

```
1. 打开 Audacity
2. 文件 → 打开 → 选择下载的音效
3. 如需剪辑：选中不需要的部分 → 删除
4. 文件 → 导出 → 导出为 MP3
5. 设置：
   - 比特率：96-128 kbps（UI音效96即可，BGM用128）
   - 声道：单声道（Mono）节省体积
6. 保存到 resources/audio/sfx/ 或 resources/audio/ambient/
```

### 3.2 音量统一

不同来源的音效音量可能不一致，建议统一处理：

```
Audacity 效果 → 标准化：
- 峰值幅度：-3.0 dB
- 应用到所有音效
```

### 3.3 文件命名检查

确保最终文件命名与代码中引用的名称一致：

```
resources/audio/
├── sfx/
│   ├── click.mp3
│   ├── hover.mp3
│   ├── confirm.mp3
```

## 四、30分钟快速采集流程

### 第1步：批量下载（10分钟）

1. 打开 Pixabay Sound Effects
2. 按关键词表逐个搜索
3. 每个关键词试听1-2个，选合适的下载
4. 预计下载 25-30 个音效（多下几个备选）

### 第2步：筛选重命名（10分钟）

1. 打开下载文件夹
2. 删除不满意的
3. 重命名为规范文件名

### 第3步：格式处理（10分钟）

1. 打开 Audacity
2. 批量导入（文件 → 导入 → 音频）
3. 统一标准化音量
4. 批量导出为 MP3（96kbps，单声道）
5. 移动到项目目录
