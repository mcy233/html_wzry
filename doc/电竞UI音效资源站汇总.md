### 1.2 免费商用资源站

#### 首推：Freesound.org（免费，需筛选）

**搜索关键词组合**:
```
esports ui
competitive game interface
electronic button click
gaming hud sound
modern ui click
digital interface sound
cyberpunk ui
```

**筛选技巧**:
1. 搜索后点击 "Filters" 
2. License 选择 "Creative Commons 0"（可商用）
3. Duration 选择 "0-2 seconds"（UI音效）
4. Sort by "Most downloaded"（质量相对有保障）

**推荐作者**:
- **LittleRobotSoundFactory** - 游戏音效专业户

## 六、音效筛选标准

下载到音效后，按以下标准筛选：

| 标准 | 合格 | 不合格 |
|------|------|--------|
| **风格** | 电子、数字、竞技感 | 卡通、真实、古典 |
| **时长** | 0.1-2秒（UI） | 超过3秒 |
| **音质** | 清晰、无杂音 | 有噪音、失真 |
| **动态** | 有冲击力、干脆 | 软弱、拖沓 |
| **频率** | 中低频为主 | 过于尖锐 |
