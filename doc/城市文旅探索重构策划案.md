# 城市文旅探索系统重构策划案

> 本策划案旨在将现有的简单"点击-领奖"城市探索升级为多维度、沉浸式的电竞+文旅体验系统。

---

## 一、现状分析

当前城市探索系统由三个文件构成：
- `js/scenes/ExploreScene.js` — 探索场景入口
- `js/data/cities.js` — 城市数据（地标 + 美食）
- `js/data/cityEvents.js` — 随机事件池

### 存在问题

| 问题 | 描述 |
|------|------|
| 交互深度不足 | 地标探索仅为"点击 → 弹出随机事件 → 获得奖励"，缺乏主动参与感 |
| 文旅属性薄弱 | 城市特色仅通过一行文字描述体现，没有视觉化、叙事化的文化展现 |
| 重玩性低 | 冷却机制使探索变成"每隔几轮点一次"的机械行为 |
| 城市无差异化 | 所有城市的交互结构完全一致，15个城市毫无区分度 |
| 美食系统简单 | 一键购买获得数值buff，无趣味性可言 |

## 二、竞品参考与借鉴

### 案例一：王者荣耀 × 苏州"东吴痛城"（2025年7-8月）

**背景**：苏州市文广旅局携手《王者荣耀》，以"东吴少年游·荣耀聚苏州"为主题，打造"苏州文旅+王者荣耀"夏日联动。

**核心机制**：
- 孙家三兄妹（孙策、孙权、孙尚香）担任"东吴荣耀推介官"，领衔三条主题体验路线
- 全城NFC碰一碰互动（超10万人次体验）
- IP主题展览盖章打卡（超4.1万人次参与兑换）
- 地铁/公交/游船换上专属东吴"皮肤"（痛列、痛车、痛船）
- 摩天轮化身"王者之眼"

**数据成效**：苏州相关景区暑期游客量同比增长35%，营业收入增长19.8%。

**借鉴点**：
1. **IP角色担任推介官** → 我们可以让KPL战队选手/吉祥物担任城市向导
2. **主题路线串联地标** → 路线化探索替代散点式点击
3. **打卡集章兑换** → 明信片/徽章收集系统
4. **全域场景沉浸** → 城市界面视觉主题化

### 案例二：Pokemon GO"Take a GO！走出趣！"（2025年9月）

**背景**：Niantic在台湾推出36条官方路线，涵盖5大主题，串联全岛文化地标。

**核心机制**：
- **5大主题分类**：文化散步、艺术之旅、风土日志、生态健行、美食巡礼
- 路线起终点收集限定明信片（10款台湾限定设计）
- 不同主题路线遇到不同主题宝可梦（含稀有异色）
- 完成路线获得橘色官方路线徽章
- 联合"岛内散步"推出10场专业导览

**借鉴点**：
1. **多主题分类路线** → 文化探源/电竞朝圣/美食地图/城市漫步
2. **明信片收集** → 每个地标一张明信片，图鉴化展示
3. **路线徽章** → 完成路线获得徽章，集齐获终极称号
4. **路线专属奖励差异化** → 不同主题路线对应不同属性提升

### 案例三：率土之滨 × 汉中文旅"北伐兴汉"（2025年5月）

**背景**：网易《率土之滨》与汉中文旅局、勉县文旅局联动，推出以诸葛亮北伐为主题的征服赛季。

**核心机制**：
- 以历史考据为蓝本，数字化重现蜀魏地理格局
- 新增"尚书令"/"副将"官职系统，融入蜀汉治理理念
- 线下景区（武侯祠、定军山）角色扮演+场景还原
- 端午节气文化融入游戏内活动

**借鉴点**：
1. **赛季叙事与城市绑定** → 客场比赛城市解锁特别探索内容
2. **历史人物活化为NPC** → 城市特色NPC互动
3. **文化知识融入游戏机制** → 城市知识问答小游戏
4. **节气/赛事周期联动** → 赛前探索提供比赛增益

## 三、重构方案详细设计

### 3.1 城市文旅主题路线系统

**设计理念**：每个城市不再是扁平的地标列表，而是提供2-3条主题路线，每条路线串联3-5个地标节点，形成有故事线的探索旅程。

**四大路线主题**：

| 主题 | 图标 | 包含内容 | 主要收益 |
|------|------|---------|---------|
| 文化探源 | 🏛️ | 历史建筑、博物馆、文物古迹 | 意识+、心态+ |
| 电竞朝圣 | 🎮 | 电竞主场、训练基地、粉丝圣地 | 士气+、粉丝+ |
| 美食地图 | 🍜 | 小吃街、名餐厅、特色市场 | 美食buff + 小游戏 |
| 城市漫步 | 🌃 | 公园、夜景、创意街区 | 状态恢复、随机奇遇 |

**路线进度机制**：
- 每条路线有进度条（如 2/4 个节点已探索）
- 不要求一次完成，可跨多个赛季轮次推进
- 完成整条路线获得：路线徽章 + 额外大奖 + 声望经验

**示例——北京路线**：
```
文化探源路线「千年帝都」：故宫 → 长城 → 南锣鼓巷 → 鸟巢
电竞朝圣路线「北京德比」：三里屯粉丝活动 → KPL北京赛区 → 北京WB训练基地
美食地图路线「京味寻踪」：全聚德烤鸭 → 胡同炸酱面 → 冰糖葫芦
```

### 3.2 城市明信片/图鉴收集系统

**明信片设计**：
- 每个地标探索后获得一张城市明信片
- 明信片包含：地标emoji图标 + 城市色 + 文化简介 + 探索日期 + 首次探索标记
- 明信片存入文旅图鉴页面

**徽章体系**：
- 集齐一个城市全部明信片 → 解锁该城市纪念徽章
- 集齐全部城市徽章 → 解锁终极称号"华夏文旅大使"
- 徽章显示在战队信息中

### 3.3 沉浸式地标交互

**交互流程升级**：
```
点击地标 → 进入沉浸式场景界面 → 阅读场景描写 → 
触发互动（NPC对话/知识问答/选择分支） → 获得奖励 + 明信片
```

**NPC系统**：每个城市2-3个特色NPC，随机出现在地标中：
- 老居民：分享城市故事，给予心态/意识加成
- 粉丝团长：组织应援活动，给予士气/粉丝加成
- 退役选手：传授经验，给予操作/配合加成
- 美食博主：推荐隐藏美食，解锁特殊美食

**知识问答**：
- 每个地标嵌入1-2道文旅知识题（答对额外奖励，答错也给知识卡片）
- 示例："故宫太和殿的原名是什么？" → A.奉天殿 B.乾清殿 C.保和殿

### 3.4 美食小游戏化

**美食匹配小游戏**：
- 进入美食环节后展示3x3食材网格
- 玩家在限时内完成食材配对（翻牌配对的简化版）
- 配对成功数决定品鉴评分（S/A/B/C）
- 评分越高，美食buff越强

**美食图鉴**：
- 记录已品尝的所有美食
- 集齐一个城市美食解锁"城市吃货"标记

### 3.5 城市声望系统

**声望等级**：每个城市独立声望值（0-5星），通过探索积累

| 星级 | 所需声望 | 解锁内容 |
|------|---------|---------|
| ★☆☆☆☆ | 0 | 基础地标可探索 |
| ★★☆☆☆ | 100 | 解锁NPC互动 + 新地标 |
| ★★★☆☆ | 300 | 解锁隐藏事件 + 折扣 |
| ★★★★☆ | 600 | 解锁城市专属道具 |
| ★★★★★ | 1000 | 荣誉市民称号 + 永久加成 |

**声望获取途径**：
- 探索地标：+20
- 完成路线节点：+30
- 完成整条路线：+100
- 品尝美食：+15
- 知识问答答对：+10

### 3.6 赛季联动叙事

- 客场比赛所在城市自动解锁"赛事特别探索"
- 赛前探索提供比赛增益（如武侯祠感悟 → 意识临时+2）
- 赛后可进行"庆功/疗伤"探索，根据比赛结果触发不同事件
- 主场城市有"主场建设"长线任务
