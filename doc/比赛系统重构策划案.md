# 比赛系统重构策划案 v4 — 事件时间轴制

> 参考《英雄联盟电竞经理》6事件时间轴 + 王者荣耀KPL真实比赛节奏  
> 版本日期：2026-04-17

---

## 一、设计目标

将原有的9轮同质化卡牌对战，重构为 **5个关键时间节点** 的策略博弈。  
每个节点对应王者荣耀比赛中的真实阶段，玩家从该节点的专属策略卡中选择一张，  
对手AI同时选择，双方揭晓后按 **三角克制关系** 和选手属性进行结算。

### 核心改进

| 维度 | v3 (旧) | v4 (新) |
|------|---------|---------|
| 比赛结构 | 9轮同质手牌出牌 | 5节点事件时间轴 |
| 决策方式 | 从手牌中选1-3张出 | 每节点从3-5张专属策略中选1张 |
| 博弈深度 | 无克制循环，靠数值堆叠 | 三角克制 + 条件策略解锁 |
| 信息透明 | 无赛前研究渠道 | 策略图鉴系统 |
| 比赛节奏感 | 轮次无差异 | 开局→发育→小龙→大龙→决胜 |

---

## 二、比赛时间轴

```
BP完成 → 开局期(0-1min) → 发育期(1-4min) → 小龙期(4-10min) → 大龙期(10-20min) → 决胜期(20min+)
```

### 节点总览

| 节点 | 名称 | 时间 | 主题 | 策略卡数量 | 是否有克制 |
|------|------|------|------|-----------|-----------|
| 1 | 开局期 | 0-1分钟 | 开局博弈 | 3 | 三角克制 |
| 2 | 发育期 | 1-4分钟 | 游走Gank | 4（+选路线） | 三角克制+安全牌 |
| 3 | 小龙期 | 4-10分钟 | 暴君争夺 | 4 | 三角克制+安全牌 |
| 4 | 大龙期 | 10-20分钟 | 主宰争夺 | 5（条件解锁） | 三角克制+无克制牌 |
| 5 | 决胜期 | 20分钟+ | 终极对决 | 3（动态根据势能） | 无固定克制 |

---

## 三、各节点详细设计

### 3.1 节点1：开局期（0-1分钟）

**场景描述**：比赛加载完毕，双方打野开始刷第一轮野怪，对线选手准备上线。是入侵对方野区、设伏等待、还是安稳发育？

#### 策略卡

| ID | 名称 | 类型 | 效果 | 代价 | 克制目标 |
|----|------|------|------|------|----------|
| open_invade | 抱团入侵 | aggression | 势能+5，偷取经济200 | 全员延迟上线 | open_normal |
| open_ambush | 集合埋伏 | defense | 势能+4，若敌方入侵则反杀(+3势能) | 全员延迟上线 | open_invade |
| open_normal | 正常上线 | economy | 经济+300，势能+2 | 无 | open_ambush |

**克制循环**：入侵 → 正常上线 → 集合埋伏 → 抱团入侵

**克制奖励**：额外势能+4，击杀+1  
**被克制惩罚**：策略效果减半，对方额外获得势能+2

**关键选手影响**：
- 打野「操作」属性影响入侵/埋伏的成功概率系数
- 辅助「意识」属性影响埋伏识别概率

---

### 3.2 节点2：发育期（1-4分钟）

**场景描述**：打野完成首轮刷野后开始游走，选择Gank哪条路线成为关键决策。需要选择策略卡 + 目标路线（上路/中路/下路）。

#### 策略卡

| ID | 名称 | 类型 | 效果 | 代价 | 克制目标 |
|----|------|------|------|------|----------|
| gank_direct | 直接Gank | aggression | 目标路线击杀概率高，势能+6 | 打野经济暂停 | gank_lure |
| gank_counter | 反蹲抓线 | defense | 反杀Gank者，势能+5 | 打野经济暂停更久 | gank_direct |
| gank_lure | 引诱抓线 | tactic | 经济+400，势能+3 | 打野经济暂停 | gank_counter |
| gank_farm | 猥琐发育 | economy | 经济+350，全员发育+8% | 放弃地图资源 | 无克制 |

**克制循环**：直接Gank → 引诱抓线 → 反蹲抓线 → 直接Gank  
**猥琐发育**无克制关系，收益稳定但偏低。

**路线选择**：
- 玩家和AI各选一条路线（上/中/下）
- 若双方选择同一路线：触发2v2冲突，克制关系生效
- 若双方选择不同路线：各自执行策略效果，不触发克制

**关键选手影响**：
- 打野「操作」影响Gank成功率
- 目标路线选手「对线」影响2v2冲突胜率

---

### 3.3 节点3：小龙期（4-10分钟）

**场景描述**：暴君（王者荣耀中的小龙）刷新，第一个关键的团队资源争夺点。是集合速杀、埋伏等待、还是趁机推线发育？

#### 策略卡

| ID | 名称 | 类型 | 效果 | 代价 | 克制目标 |
|----|------|------|------|------|----------|
| drake_rush | 速杀暴君 | objective | 获得暴君buff(势能+7)，伤害+5% | 经济发育暂停 | drake_flank |
| drake_ambush | 蹲伏暴君 | defense | 伏击成功势能+6，击杀+2 | 发育暂停更久 | drake_rush |
| drake_flank | 包抄暴君 | tactic | 经济+500，势能+4 | 可能丢失暴君 | drake_ambush |
| drake_farm | 猥琐发育 | economy | 经济+400，势能+2 | 放弃暴君 | 无克制 |

**克制循环**：速杀 → 包抄 → 蹲伏 → 速杀

**暴君buff**：获取后在后续节点中持续提供 伤害+5% 加成

**关键选手影响**：
- 全队「配合」平均值影响团战执行力
- 打野「意识」影响争龙smite成功率

---

### 3.4 节点4：大龙期（10-20分钟）

**场景描述**：主宰（王者荣耀中的大龙）刷新，比赛进入中后期的关键转折点。争夺大龙还是趁机推进？

#### 策略卡

| ID | 名称 | 类型 | 效果 | 代价 | 克制目标 | 解锁条件 |
|----|------|------|------|------|----------|----------|
| baron_rush | 速杀主宰 | objective | 获得主宰buff(势能+10)，全线推进 | 经济发育暂停 | baron_flank | 始终可用 |
| baron_ambush | 蹲伏主宰 | defense | 伏击成功势能+8，击杀+3 | 发育暂停更久 | baron_rush | 始终可用 |
| baron_flank | 包抄主宰 | tactic | 经济+600，势能+5 | 可能丢失主宰 | baron_ambush | 始终可用 |
| baron_push | 抱团推塔 | aggression | 推塔势能+6，经济+300 | 放弃主宰争夺 | 无克制 | 始终可用 |
| baron_split | 四一分推 | tactic | 单带势能+5，团队防御+10% | 放弃主宰争夺 | 无克制 | 对抗路选手rating>=85 |

**克制循环**：速杀 → 包抄 → 蹲伏 → 速杀（推塔/分推无克制）

**主宰buff**：获取后在决胜期提供 势能+5 加成和全线兵线优势

**关键选手影响**：
- 全队「配合」+「意识」平均值影响团战
- 对抗路「对线」影响分推效果

---

### 3.5 节点5：决胜期（20分钟+）

**场景描述**：比赛进入最终阶段。根据当前势能不同，可用的策略卡不同。

#### 顺风策略（势能 >= 60）

| ID | 名称 | 效果 |
|----|------|------|
| final_end | 终结比赛 | 五人推中，推塔势能+12 |
| final_steady | 稳步推进 | 安全推进，势能+6，经济+400 |
| final_push | 抱团推塔 | 选择一路推进，势能+8 |

#### 均势策略（40 < 势能 < 60）

| ID | 名称 | 效果 |
|----|------|------|
| final_teamfight | 发起团战 | 触发最终团战结算（基于全队属性） |
| final_push_mid | 抱团推中 | 推中路，势能+7 |
| final_defend | 防守反击 | 防御+15%，等待对方失误 |

#### 逆风策略（势能 <= 40）

| ID | 名称 | 效果 |
|----|------|------|
| final_desperate | 绝地反击 | 全员伤害+130%，无法撤退，胜率35% |
| final_backdoor | 偷水晶 | 高风险高回报，成功率25%直接获胜 |
| final_highground | 防守高地 | 防御+30%，拖延时间等翻盘机会 |

**决胜期特殊机制**：
- 若前4节点积累了暴君/主宰buff，此处额外加成
- AI根据势能差距选择激进或保守策略
- 若双方势能接近（差距<5），触发"终极团战"子系统：基于全队6维属性加权随机决胜

**关键选手影响**：
- 全队「心态」+「抗压」决定逆风翻盘系数
- MVP候选人在决胜期获得额外加成

## 四、结算系统

### 4.1 势能（Momentum）

- **初始值**：50 + BP阶段势能加成（-10 ~ +10）
- **每节点结算**：根据策略效果、克制结果、选手加成调整
- **胜负判定**：
  - 任意节点后势能 >= 75 → 提前胜利（压倒性优势）
  - 任意节点后势能 <= 25 → 提前失败
  - 5节点全部完成后 势能 >= 50 → 胜利，< 50 → 失败

### 4.2 克制结算公式

```
baseEffect = 策略卡基础效果
playerBonus = 相关选手属性加成 (0 ~ +30%)
counterBonus = 克制成功 ? +4势能 + 击杀+1 : 0
counterPenalty = 被克制 ? 效果×0.5 + 对方+2势能 : 0
sameStrategy = 双方相同 ? 各自获得正常效果的70% : 0

节点势能变化 = baseEffect × (1 + playerBonus) + counterBonus - counterPenalty
```

### 4.3 经济系统

- 每节点基础经济增长：+500
- 策略卡附加经济（如入侵偷经济、发育增加经济等）
- 经济差每 2000 → ±1 势能修正（在节点结算后附加）
- 经济数据仅用于势能修正和赛后统计展示

### 4.4 击杀系统

- 克制成功：+1 击杀
- Gank成功（发育期路线冲突）：+1~2 击杀
- 伏击成功：+2 击杀
- 团战胜利：+2~3 击杀
- 击杀每3个 → +1 势能

### 4.5 选手属性加成

各属性在不同节点的权重：

| 属性 | 开局期 | 发育期 | 小龙期 | 大龙期 | 决胜期 |
|------|--------|--------|--------|--------|--------|
| 操作 | 打野×1.5 | 打野×1.2 | 0.5 | 0.5 | 0.8 |
| 意识 | 辅助×1.2 | 0.8 | 全队×1.0 | 全队×1.0 | 0.6 |
| 对线 | 0.3 | 目标路线×1.5 | 0.3 | 对抗路×1.2 | 0.3 |
| 配合 | 0.5 | 0.5 | 全队×1.5 | 全队×1.5 | 1.0 |
| 抗压 | 0.3 | 0.3 | 0.5 | 0.5 | 全队×1.5 |
| 心态 | 0.3 | 0.3 | 0.5 | 0.8 | 全队×1.5 |

加成计算：`playerBonus = Σ(属性值 - 75) × 权重 × 0.003`  
即属性高于75才有正加成，低于75产生负加成。

## 五、AI决策逻辑

### 5.1 风格系统

延续战队风格影响AI倾向：

| 风格 | 偏好 | 权重调整 |
|------|------|----------|
| 激进进攻型 | 入侵、直接Gank、速杀 | aggression卡+30%概率 |
| 运营碾压型 | 发育、包抄、推塔 | economy/tactic卡+30% |
| 团战碾压型 | 速杀、团战、终结 | objective卡+30% |
| 防守反击型 | 埋伏、反蹲、防守 | defense卡+30% |

### 5.2 势能感知

- 己方势能 > 60：倾向选择进攻策略
- 己方势能 < 40：倾向选择安全策略
- 均势：按风格基础概率

## 六、策略图鉴系统

### 6.1 功能定位

赛前研究工具，让玩家在进入比赛前可以查看所有策略卡的效果和克制关系。

### 6.2 界面布局

```
┌──────────┬──────────────────┬──────────────────┐
│ 节点导航  │  策略卡网格(3-5张) │   卡牌详情面板    │
│          │                  │                  │
│ ● 开局期  │ [卡1] [卡2] [卡3] │  名称 / 描述      │
│ ○ 发育期  │                  │  效果 / 代价      │
│ ○ 小龙期  │                  │  克制关系图       │
│ ○ 大龙期  │                  │  选手属性影响     │
│ ○ 决胜期  │                  │  战术建议        │
└──────────┴──────────────────┴──────────────────┘
```

### 6.3 入口

- HomeScene 主菜单："策略图鉴" 按钮
- BP完成后结果页面："查看策略" 链接

## 八、UI/UX设计要点

### 8.1 节点决策界面

- 顶部：时间轴进度条（5个节点，高亮当前节点）
- 左侧：当前场景描述 + 势能/经济/击杀面板
- 中间：策略卡选择区（卡片展示效果、代价、克制关系）
- 右侧：AI教练建议面板
- 底部：确认选择按钮

### 8.2 结算动画

- 双方策略卡同时翻转揭晓
- 克制关系以箭头动画展示（A→B 表示A克制B）
- 势能条动态增减
- 击杀/经济变化飘字
- 若提前判定胜负，播放特殊动画

### 8.3 比赛结果页

沿用现有的胜利/失败展示，增加：
- 5节点决策回顾时间轴
- 每节点的策略对比和克制结果
- MVP评选（基于节点贡献）
