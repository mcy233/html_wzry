# 美术资源采集清单

> 本清单列出游戏所需的全部美术资源，每项附带AI图片生成提示词。
> 
> **输出规格统一要求**：
> - 背景图：**1920×1080**，WebP格式，质量75-80
> - 卡片/图标：指定尺寸，WebP格式，质量80-85
> - 风格基调：**暗色科技电竞风**，主色 `#0a0e17` 深空黑 + `#4ea8de` 电竞蓝 + `#f0c040` 竞技金
> 
> **文件保存位置**：全部放入 `resources/` 对应子目录

## 三、音频资源

### 3.1 BGM（3段）

保存至 `resources/audio/bgm/`

> **更快的方式**：推荐直接从免费音乐库获取，比AI生成质量更好：
> - **Mixkit.co**（免费商用）
> - **Incompetech.com**（CC BY署名即可）
> - **Pixabay Music**（免费）
>
> 搜索关键词如下：

| 文件名 | 搜索关键词 | 时长 | 要求 |
|--------|-----------|------|------|
| `menu.mp3` | `esports menu theme`, `cyberpunk ambient`, `electronic chill` | 60-120秒循环 | 节奏舒缓，有科技感，适合长时间聆听不烦躁 |
| `battle.mp3` | `epic battle music`, `competitive gaming`, `intense electronic` | 90-180秒循环 | 节奏紧张，鼓点有力，适合对战氛围 |
| `victory.mp3` | `victory fanfare`, `triumph orchestral`, `winning celebration` | 15-30秒 | 短促激昂，号角/管弦+电子，胜利时播放 |

**格式要求**：MP3，96kbps，单声道（节省体积），循环曲需确保首尾可无缝衔接

| `crowd-cheer.mp3` | `crowd cheering esports`, `audience applause arena` | 10-15秒循环 | 赛场观众欢呼，胜利时或关键时刻播放 |
| `arena-hum.mp3` | `arena ambience`, `indoor crowd murmur` | 30-60秒循环 | 赛场底噪，BP阶段低音量循环播放 |

## 六、采集完成后的集成工作

> 以下步骤由开发者（AI）完成，你只需把资源放到对应目录：

| 资源放入的目录 | 集成工作（开发者完成） |
|--------------|---------------------|
| `resources/bg/*.webp` | 在CSS中引用为各场景背景图 |
| `resources/bg/cities/*.webp` | ExploreScene中按城市名加载 |
| `resources/ui/game-logo.*` | 加载界面和标题界面引用 |
| `resources/icons/*.svg` | 替换现有emoji为SVG图标 |
| `resources/ui/corner-deco.svg` | CSS中引用为面板装饰 |
| `resources/audio/bgm/*.mp3` | SoundManager新增MP3播放集成 |
| `resources/audio/ambient/*.mp3` | SoundManager新增氛围音播放 |
| `resources/heroes/*.webp` | heroes.js数据关联 + BP界面引用 |
