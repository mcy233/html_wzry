## 图片规格

| 类型 | 推荐尺寸 | 格式 | 说明 |
|------|---------|------|------|
| 战队Logo | 200×200px | PNG (透明背景) | 文件名 = teams.js 中的 id |
| 选手头像 | 200×200px | PNG/JPG | 文件名 = 选手游戏ID（如「暖阳」） |

## 获取方式

### 战队Logo
1. 打开 https://kpl.qq.com/#/Teams
2. F12 → Network → 筛选 Img
3. 刷新页面，找到战队logo图片
4. 右键 → 另存为 → 按上面命名保存到 `resources/teams/`

### 选手定妆照/头像
1. 在 KPL官网 点击某个战队进入详情
2. 找到选手头像图片
3. 右键 → 另存为 → 按选手ID命名保存到 `resources/players/`

### 替代来源
- 微博搜索「KPL 2026春季赛 定妆照」
- 百度图片搜索「KPL 战队logo 高清」
- 虎扑王者荣耀版块

## 注意事项
- 文件名必须与代码中的ID完全一致（区分大小写）
- 放入图片后刷新游戏即生效，无需修改任何代码
- 支持 PNG 和 JPG 格式
